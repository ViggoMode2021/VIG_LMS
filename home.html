This app was created with Python/Flask, as well as PostgreSQL/psycopg2. It is a gradebook that reinforces CRUD (Create, Read, Update, Delete). The 'create' allows students to be enrolled, 'read' allows the user to see the class roster, 'update' allows the user to update a student's grade, and 'delete' allows the user to unenroll a student from a class. Please use the navbar above to navigate through the app! Thank you! - Ryan Viglione
